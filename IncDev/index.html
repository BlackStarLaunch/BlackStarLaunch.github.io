Object Oriented Programming Using Black Culture Analogies

Learning to code? It’s best to make sure you have a solid foundation and thoroughly understand object-oriented programming concepts such as objects, classes, inheritance and interfaces. We’ll explore these concepts in detail using easier to understand analogies. Because this course is specifically designed for technologists of color — we ditch the Star Wars/Dungeons & Dragons analogies for engaging and playful parallels to black culture staples.
